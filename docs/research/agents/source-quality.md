# Agent Source Quality Policy

All MentorOS Agent materials are research summaries for cognitive-model design. They are not biographies, quote banks, or impersonation scripts.

## Reliability Levels

- A: official sites, public speeches hosted by institutions, publisher pages, official archives, verified primary material.
- B: reputable media, edited interviews, credible transcript mirrors, public compilations with clear provenance.
- C: blogs, social posts, reader notes, secondary explainers. Use only for orientation and cross-checking.

## Verification Status

- verified: the implementer opened and checked the page during implementation.
- needs_verification: plausible public source, but not fully checked in this pass.
- draft: useful pointer or known public discourse, but source chain still needs cleanup.

## Use Rules

- Prefer summaries, tags, and structured observations over source prose.
- Do not import full books, paid courses, pirated PDFs, or long copyrighted excerpts.
- Do not fabricate quotes or write quote-like attributions without a future citation system.
- If a source is C-level or draft, it may shape hypotheses but should not anchor hard claims.
