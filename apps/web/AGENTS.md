# apps/web Rules

This workspace owns the static frontend prototype, visual design, interactions, and mock data.

## Allowed

- Edit files inside `apps/web`.
- Use mock data from `apps/web/lib/mock-data.ts`.
- Build static pages and visual components.

## Forbidden

- Do not call OpenAI or any model provider.
- Do not read or write Supabase.
- Do not add backend routes.
- Do not implement auth.
- Do not edit `packages/ai`, `packages/db`, `packages/agents`, or `packages/evals`.

## Visual Direction

Use **A. Quiet Study**: black-white-gray, large whitespace, quiet sidebar, reading-width conversation column, and a lightweight fixed bottom input.
